# Space Harrier-like (Pseudo-3D Shooter)

Shooter sobre raíles con profundidad simulada. Vuela hacia el horizonte destruyendo enemigos que se acercan, esquiva proyectiles y derrota a los jefes de etapa.

## Controles

| Tecla | Acción |
|-------|--------|
| ← → | Moverse izquierda/derecha |
| ↑ ↓ | Moverse arriba/abajo |
| J / Espacio | Disparar (mantener para carga) |

### Gamepad
- LStick: Moverse
- A / R2: Disparar

## Mecánica

- **Pseudo-3D**: los enemigos aparecen en el horizonte y crecen al acercarse.
- **Auto-fire**: mantén disparo para fuego continuo. El power level aumenta el número de balas.
- **Ataque cargado**: mantén disparo 1.5s para un blast que daña todos los enemigos en frente.
- **Power-ups**: los enemigos sueltan power-ups (P) que aumentan tu nivel de disparo (max 5).
- **3 etapas**: cada una con tema visual y jefe diferente.

## Escenarios

| Etapa | Tema | Dificultad |
|-------|------|------------|
| 1 | Cielo y pradera 🌤️ | Normal |
| 2 | Desierto al atardecer 🌅 | Difícil |
| 3 | Noche espacial 🌌 | Muy difícil |

## Logros

- **Primer vuelo**: completa la etapa 1
- **As del espacio**: completa las 3 etapas
- **Guerrero del cielo**: llega a la etapa 3
- **Vuelo perfecto**: termina con 3 vidas
