# Street Fighter-like (Pelea 1v1)

Juego de lucha 1v1 local (PvP) o contra IA con 4 personajes seleccionables.

## Controles

### Jugador 1 (WASD + JKLU)
| Tecla | Acción |
|-------|--------|
| W / A / S / D | Arriba / Izquierda / Abajo / Derecha |
| J | Puñetazo |
| K | Patada |
| L | Especial |
| U | Super (requiere barra llena) |

### Jugador 2 (Flechas + RF TY)

| Tecla | Acción |
|-------|--------|
| ↑ ← ↓ → | Arriba / Izquierda / Abajo / Derecha |
| R | Puñetazo |
| F | Patada |
| T | Especial |
| Y | Super (requiere barra llena) |

### Generales
| Tecla | Acción |
|-------|--------|
| Espacio / Enter | Seleccionar / Siguiente ronda |
| Escape | Volver |

### Gamepad
- Soporte completo para 2 gamepads (P1 y P2). Las acciones se mapean automáticamente a los bindings por jugador.

## Mecánica

- **Al mejor de 3 rounds**: el primer jugador en ganar 2 rounds gana el combate.
- **Super meter**: se llena al atacar y recibir daño. Al alcanzar el máximo, se puede usar un super (más daño).
- **Bloqueo**: si el oponente se mueve hacia atrás mientras recibe un ataque, bloquea parcialmente (daño reducido).
- **4 personajes**: cada uno con stats únicos (HP, velocidad, daño) y un super especial.
- **IA adaptativa**: la IA tiene 3 niveles de dificultad y usa lógica de decisión basada en distancia al oponente.
