# Contra-like (Run & Gun)

Run & gun con scroll lateral automático, power-ups de armas, oleadas de enemigos y jefes finales.

## Controles

| Acción | Teclado | Gamepad |
|--------|---------|---------|
| Moverse | WASD / Flechas | LStick |
| Saltar | Espacio / K | GamepadX |
| Disparar | J / Click | GamepadA (automático si se mantiene) |
| Cambiar arma | L | GamepadY |
| Reiniciar | Espacio | GamepadStart / GamepadA |

## Armas

- **DEFAULT**: Pistola estándar, cadencia media
- **S**: Spread (3 balas en abanico)
- **M**: Machine Gun (alta cadencia)
- **L**: Laser (daño doble, penetrante)
- **F**: Fire (daño triple, 2 balas)

## Power-ups

Caen de enemigos al destruirlos (20% de probabilidad). Recoge las letras para cambiar de arma.

## Progresión

- 3 etapas con dificultad creciente
- Jefe final en cada etapa con patrones de ataque únicos
- 3 vidas, puntuación con récord persistente
- Logros: completar etapa 1, completar el juego, alcanzar etapa 3
