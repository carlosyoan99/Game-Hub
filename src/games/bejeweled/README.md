# 💎 Bejeweled (Match-3 Puzzle)

Intercambia gemas adyacentes para formar líneas de 3 o más.
¡Cascadas, gemas especiales y 3 modos de juego!

---

## Controles

| Tecla / Ratón | Acción |
|---------------|--------|
| **Click** en una gema | Seleccionar gema |
| **Click** en gema adyacente | Intercambiar gemas |
| `← → ↑ ↓` | Navegar cursor (teclado) |
| `Space` / `Enter` | Confirmar intercambio / Seleccionar modo |

## Modos de juego

| Modo | Objetivo | Límite |
|------|----------|--------|
| **Clásico** 🎯 | Alcanza 2000 puntos | 20 movimientos |
| **Contrarreloj** ⏱ | Alcanza 3000 puntos | 90 segundos |
| **Infinito** ♾ | Juega sin límites | Sin límite |

## Gemas especiales

| Match | Gema especial | Efecto |
|-------|---------------|--------|
| 3 iguales | — | Gemas eliminadas, puntuación base × multiplicador |
| 4 iguales | **Rayada** 🟡 | Aparece con borde dorado + estrella |
| 5 iguales | **Bomba** ★ | Destello central, elimina área más grande |

## Mecánicas

- **Cascadas**: Después de un match, las gemas caen llenando vacíos.
  Nuevas gemas aparecen desde arriba, pudiendo crear más matches.
- **Multiplicador de cascada**: Cada cascada consecutiva aumenta el
  multiplicador (x1 → x2 → x3 ... x8).
- **Sin movimientos válidos**: El tablero se revuelve automáticamente.
- **Selección visual**: La gema seleccionada brilla con borde dorado.
- **Cursor teclado**: Aparece un borde punteado que puedes mover.

## Logros

| Logro | Condición |
|-------|-----------|
| ⭐ Primera victoria | Completa un modo de juego |
| 🏆 Maestro de cascadas | Alcanza combo de 5 cascadas |
| 👑 Acaparador de gemas | Limpia 100 gemas en total |
| 👑 Demonio de velocidad | Gana en modo Contrarreloj |
