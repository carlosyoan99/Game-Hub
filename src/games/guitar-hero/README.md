# 🎸 Guitar Hero (Juego de Ritmo)

Toca las notas que caen por la pantalla con los dedos correctos.
¡Mantén el combo, acumula Star Power y conviértete en una leyenda del rock!

---

## Modos de juego

### 🎯 Normal
- HP que decae con el tiempo, se recupera al acertar notas
- Game Over si HP llega a 0
- Puntuación, combo, Star Power
- Progresión guardada (logros, XP, stats)

### 🆕 Práctica
- **Sin HP**: no hay game over, falla las notas que quieras
- **Velocidad ajustable**: 50%, 75% o 100% con `↑`/`↓`
- Sin límite de tiempo ni presión
- Ideal para aprender canciones nuevas
- No guarda progresión (practica sin consecuencias)

## Mecánicas

- **Puntuación**: Perfect (+300), Good (+100), Miss (pierde combo + HP)
- **Combo**: 5+ notas seguidas activa multiplicador (x2, x3, x4, x5)
- **Star Power**: Se acumula al hacer combos. Actívala con `Space` en combo (gasta 50%)
  - Duplica el multiplicador durante 5 segundos
  - Ilumina la pantalla con un brillo dorado
- **HP**: Se pierde al fallar notas, se gana al acertar. Si llega a 0, GAME OVER
- **Acordes**: Notas dobles simultáneas (marcadas con borde dorado)
- **Canciones procedurales**: Cada partida genera una secuencia única
