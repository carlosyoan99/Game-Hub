# Super Mario Bros

Un juego tipo Super Mario con scroll lateral, power-ups y enemigos clásicos.

## Mecánicas

- **Movimiento**: corre y salta a través de niveles con scroll lateral.
- **Power-ups**: recoge hongos para crecer y flores para disparar bolas de fuego.
- **Enemigos**: Goombas (saltar encima para aplastar) y Koopas (patada de caparazón).
- **Bloques**: ? bloques dan monedas o power-ups; ladrillos se rompen si eres grande.
- **Monedas**: 100 puntos cada una.
- **3 niveles** de dificultad progresiva, con bandera de meta al final.

## Controles

| Acción | Teclado | Gamepad |
|--------|---------|---------|
| Moverse | ←/→, A/D | LStick L/R, D-pad L/R |
| Saltar | Espacio, ↑, W | GamepadA |
| Correr | Shift | GamepadX |
| Disparar (con flor) | Espacio | GamepadB |
| Reiniciar | Espacio | GamepadStart / GamepadA |

## Power-ups

| Power-up | Efecto |
|----------|--------|
| 🍄 Hongo | Te hace grande (2 hits para morir) |
| 🔥 Flor | Disparas bolas de fuego |
| ⭐ Estrella | Invulnerabilidad temporal |

## Logros

- `world-clear`: Completa cualquier nivel
- `castle-clear`: Completa los 3 mundos
