# 🪓 Golden Axe (Beat'em Up)

Hack & slash lateral con scroll automático. Elige entre 3 guerreros,
combate oleadas de enemigos, usa magia y derrota jefes para avanzar.

---

## Controles

| Tecla | Acción |
|-------|--------|
| `A/D` o `← →` | Moverse |
| `W` / `↑` | Saltar |
| `J` | Atacar (cuerpo a cuerpo) |
| `K` | Ataque especial (gasta magia) |
| `L` | Magia (área, gasta toda la magia) |
| `Space` | Saltar |
| `Enter` / `Space` | Confirmar/Continuar |

## Mecánicas

- **Combos**: Golpea enemigos seguido (máximo 5x). El combo se reinicia tras 1.5s sin golpear.
- **Magia**: Se recupera con pociones azules. Gasta 8-10 MP para ataque especial, o toda la magia para ataque de área devastador.
- **Pociones**: Caen de enemigos (15% de probabilidad). Azules = magia, Verdes = HP.
- **3 Etapas**: Cada etapa tiene un jefe al final con patrón único (spread, charge, spiral).
- **Auto-scroll**: La pantalla avanza automáticamente. El jugador no puede quedarse atrás.
